## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/shreekantha-portfolio.git
```

Go to the project folder

```bash
cd shreekantha-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
